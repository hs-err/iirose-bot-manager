=========使用说明=========
安装上就可以了~
math支持极其复杂的数学计算~
（调用maxima实现）
注意末尾的分号~
内存限制：100M
cpu 限制：0.1 core
时间限制：5秒
=========使用示例=========
/math 1+1;
解释：计算1+1;
(%i1) 1+1;
(%o1)                                  2

/math solve(x^2+1=5,x);
解释：解方程x^2+1=5中的x
(%i1) solve(x^2+1=5,x);
(%o1)                          [x = - 2, x = 2]

/math diff(x^3+x^2,x,1);
解释：一次求导x^3+x^2中的x
(%i1) diff(x^3+x^2,x,1);
                                     2
(%o1)                             3 x  + 2 x
