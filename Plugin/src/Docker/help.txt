=========使用说明=========
安装上就可以了~
docker为每人一个用户，默认在主目录
支持所有指令，如需使用sudo请联系logos或苏苏
docker内存限制：1G
docker cpu限制：1 core
docker 系统：ubuntu
docker 环境：php7.2 nodejs npm
docker 时限：5秒
=========使用示例=========
/docker ping www.baidu.com
解释：在docker内执行ping www.baidu.com
